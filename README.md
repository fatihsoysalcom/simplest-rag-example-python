# Simplest RAG Example Python

This example demonstrates the core concept of Retrieval Augmented Generation (RAG) using a simple, self-contained Python script. It shows how external knowledge can be retrieved and used to augment an LLM's prompt, leading to more accurate and contextually relevant answers compared to a 'naive' LLM. The script includes a mock LLM and a basic keyword-based retrieval system.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
